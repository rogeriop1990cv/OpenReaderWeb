# Open Reader Web

Criar um site para ler mangas e novels online.

## Tecnologias

Frontend:
    - [React](https://reactjs.org/)
    - Material-UI
    - Tailwind CSS
    - TypeScript

Backend:
    - Node.js
    - Express
    - TypeScript
    - MySQL
    Orm:
        - Sequelize
        - Prisma

## Funcionalidades

- Ler mangas e novels online
- Favoritar mangas e novels
- Histórico de leitura
- Comentários
